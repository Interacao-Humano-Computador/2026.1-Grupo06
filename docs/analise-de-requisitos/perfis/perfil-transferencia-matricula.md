## Introdução
O perfil de usuário é uma descrição detalhada das características dos indivíduos cujas atividades o sistema visa apoiar. Para isso, devem-se identificar os atributos de interesse (cargo, função, experiência, nível de instrução, atividades principais, faixa etária, etc.). O objetivo de estudar o perfil do usuário é facilitar e assegurar a qualidade na criação de personas, garantindo, consequentemente, maior excelência no desenvolvimento do produto.

Para esta funcionalidade foi utilizada a técnica de revisão e análise de documentos

## Metodologia
A razão da escolha desta metodologia foi a facilidade de realização e a possibilidade de se obter dados históricos de usuários e também expectativas criadas ao redor do produto encontradas em notícias, machetes e análises já feitas anteriormente. Por se tratar de uma funcionlidade ainda não existente, é possível entender por meio de documentos a forma como o processo ocorre atualmente, ou seja, sem a utilização do sistema.
