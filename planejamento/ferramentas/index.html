  Para a realização do projeto serão utilizadas algumas ferramentas para facilitar a organização da equipe e a produção dos artefatos. As ferramentas escolhidas para a execução do projeto estão apresentadas na Tabela 1.

Justificativa das Ferramentas

  As ferramentas foram escolhidas com o objetivo de apoiar diferentes etapas do projeto, incluindo organização, comunicação, desenvolvimento e análise. Ferramentas como GitHub e MkDocs foram utilizadas para documentação e versionamento, enquanto plataformas como Microsoft Teams e WhatsApp facilitaram a comunicação entre os integrantes.

  Além disso, ferramentas como WAVE foram utilizadas para análise de acessibilidade, e o uso de inteligência artificial, como o ChatGPT, auxiliou na organização dos conteúdos e no apoio à análise das tarefas. A técnica de análise de tarefas foi aplicada para compreender as ações dos usuários e identificar possíveis dificuldades na interação com o sistema.
